# AI Resume Match Score

## Description
An AI-based Resume Match Score application that compares a resume with a job description and calculates a matching score.

## Features
- Resume Parsing
- Job Description Analysis
- Resume Match Score
- NLP using NLTK
- Python-based Project

## Technologies Used
- Python
- NLTK
- Flask
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository.
2. Install dependencies.
3. Run the application.

```bash
python app.py
```
